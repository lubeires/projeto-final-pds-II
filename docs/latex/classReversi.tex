\doxysubsection{Detailed Description}
Classe que representa o jogo \doxylink{classReversi}{Reversi} (também conhecido como Othello). 

Esta classe é uma implementação do jogo de tabuleiro \doxylink{classReversi}{Reversi}, que é jogado em um tabuleiro 8x8. O objetivo do jogo é capturar as peças do adversário ao cercá-\/las com suas próprias peças. A classe gerencia as jogadas, verificações de vitória, e captura de peças durante o jogo.

\begin{DoxyNote}{Note}
Esta classe herda de {\ttfamily \doxylink{classJogoDeTabuleiro}{Jogo\+De\+Tabuleiro}}, que define métodos genéricos para jogos de tabuleiro. 
\end{DoxyNote}
